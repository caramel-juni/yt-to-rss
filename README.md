# yt-to-rss

a small python script that converts the `subcriptions.csv` part a youtube takeout export into an RSS reader importable OPML feed.

### Usage:

1. `git clone https://github.com/caramel-juni/yt-to-rss/`
2. `cd yt-to-rss/`
3. `python3 yt-to-rss.py subscriptions.csv subscriptions.opml`
4. Import generated `subscriptions.opml` into RSS reader of choice! :3

*inspired by this [article](https://charlesthomas.dev/blog/converting-my-youtube-subscriptions-to-rss-2025-03-01/)*
